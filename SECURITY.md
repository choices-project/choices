# Security Policy

_Documentation last reviewed: April 4, 2026._

## Reporting a Vulnerability

Please email `security@choices-app.com` with details. We prefer an initial non‑public report. Include steps to reproduce, impact, and any suggested remediation.

We will acknowledge within 5 business days and provide a timeline for remediation once assessed.

## Scope

This repository and the hosted Choices application.

## Safe Harbor

If you follow this policy in good faith, we will not pursue legal action against you for your research. Avoid accessing user data beyond what is necessary to demonstrate the issue.

## Technical documentation

For architecture, rate limiting, RLS, and implementation detail, see [docs/SECURITY.md](docs/SECURITY.md).
